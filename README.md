# Spring Boot REST Service

## Description

A simple Spring Boot REST Service

## Installation

1. [Install Gradle](https://gradle.org/install/)

```bash
# MacOS
brew install gradle
```

2. Clone this repo to your local machine

```bash
git clone https://github.com/dominicgaliano/payroll-spring-boot/
cd gs-rest-service/
```

3. Run the build

```bash
./gradlew bootRun
```
